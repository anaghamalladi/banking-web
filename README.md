# PyBank — Banking Web App

A full-stack banking web application built with Python Flask and SQLite.

## Live Features
- User signup and login with password hashing
- Deposit, withdraw, and transfer money between accounts
- Auto-generated 10-digit account numbers
- Spending breakdown doughnut chart
- 7-day activity bar chart
- Transaction filter by type
- Low balance and large transaction notifications

## Tech Stack
- **Backend:** Python, Flask, SQLite
- **Frontend:** HTML, CSS, JavaScript, Chart.js

## How to Run
```bash
pip3 install flask
python3 app.py
```
Then open http://127.0.0.1:5000
